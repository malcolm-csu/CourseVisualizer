# Student File Template

Copy one of the blocks below, fill it in, save as `lastname_firstname.txt`, then run:

```bash
./run.sh lastname_firstname.txt --open
```

---

## Text format

```
Name: Last, First
ID: 888000000
Degree: BSCS

# Lower Division
MAT 153
CSC 115
CSC 121

# Add completed courses above — one per line
# Lines starting with # are ignored
```

---

## Degree codes

BSCS · BSIT · BAITHS · BAITG · BAITP · MinorCS · MinorIT · MSCSDSN · MSCSSE · CertIT
